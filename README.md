# agrinho2026isabella 
preciso de um codigo html e style para o agrinho 2026 no parana com o tema  Landing Page: Guardiões da Polinização
O que é: Um site de página única focado na importância das abelhas e polinizadores para a produtividade da soja e do café no Paraná, mostrando como a sustentabilidade protege esses insetos.
Recursos de HTML/CSS: Menu de navegação fixo (position: fixed), uso de flexbox ou grid para criar uma galeria de fotos das espécies nativas e rodapé com um formulário de "Compromisso Sustentável" estilizado.
